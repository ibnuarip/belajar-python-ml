# Belajar Python & Machine Learning

Project belajar Machine Learning menggunakan Python dan scikit-learn.

## Yang Dipelajari
- Setup Python & virtual environment
- Exploratory Data Analysis (EDA) dengan pandas
- Membersihkan dan memproses dataset
- Melatih model Linear Regression
- Evaluasi model (R2 Score, MAE)
- Visualisasi data dengan matplotlib
- Jupyter Notebook

## Dataset
Student Performance Factors dari Kaggle

## Tech Stack
- Python 3.14
- scikit-learn
- pandas
- matplotlib
- Jupyter Notebook

## Cara Menjalankan
1. Clone repo ini
2. Buat virtual environment
```
   python -m venv venv
```
3. Aktifkan virtual environment
```
   source venv/Scripts/activate  # Windows Git Bash
```
4. Install library
```
   pip install -r requirements.txt
```
5. Jalankan script
```
   python model.py
```
